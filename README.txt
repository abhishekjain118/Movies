## this is a readme file.
## it tells how to open the fresh_tomatoes.html file. 

1. you need python installed on your PC to run this application. 
2. download the zip file. Unzip it. 
3. go to unzipped Folder, run the python file entertain_center.py 
4. the file entertain_center.py will generate an HTML webpage, fresh_tomatoes.html
5. open fresh_tomatoes.html in a browser. 
6. Click on any of the movie poster to see it's trailer.
